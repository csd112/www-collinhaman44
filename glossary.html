Glossary

HTML
Stands for hypertext markup language. It allows for the creation of websites using different tools.
Character sets
Character sets allow you to use symbols from different languages in your html documents.
Favicon
A favicon displays an icon next to an open tab for your website. useful for displaying your brand logo.
Hyperlink
A hyperlink is a link on a website that if clicked on can lead you to another website or a different part of the website you are currently on.
Thumbnail
A smaller sized copy of an image used to represent the larger version of that image.
